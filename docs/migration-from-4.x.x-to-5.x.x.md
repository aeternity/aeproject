# Migration from 4.x.x to 5.x.x

## Changes

**AEproject** `v5.0.0` updated the sdk to the latest version, see the [migration guide](https://github.com/aeternity/aepp-sdk-js/blob/v14.0.0/docs/guides/migration/14.md) for additional reference.

Install the new AEproject version

```
npm install -g @aeternity/aeproject
```

## Instructions

1. Upgrade your project

   ```
   aeproject init --update
   ```
2. Adapt Testing Setup using the [migration guide](https://github.com/aeternity/aepp-sdk-js/blob/v14.0.0/docs/guides/migration/14.md) provided by the sdk.
